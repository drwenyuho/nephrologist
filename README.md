## 文章更新流程
1. 先預覽部落格樣式
```powershell title='terminal'
npm start
```
2. 預覽確認無誤，依下列指令推上 github:
```powershell
git add .
```
```powershell
git commit -m "你的紀錄"
```
```powershell
git push
```

## 若我有更新程式碼
那請先依照下列步驟更新你本地專案內容。

1. 更新程式碼
```powershell
git pull origin main
```
:::warning
這次需要輸入下面指令。
:::
2. 這一項非必須，如有需要，我會做紅色警告註解在此項目上方。
```powershell
npm install
```
